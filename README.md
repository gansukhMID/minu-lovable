## Setup

1. **Clone & Install**

```bash
git clone <your-fork-or-repo-url>
cd minu-lovable
pnpm install
```

2. **Configure `.env.local`** — see `.env.example` (sandbox + AI keys).

3. **Run**

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000).
