# Task: Rewrite generate-ai-code-stream as module assembler

## Description

Replace the current arbitrary code-generation logic in `app/api/generate-ai-code-stream/route.ts` with a module assembler. The new assembler accepts a list of validated module names and an `InstanceConfig`, then streams the assembly of the customer-web from pre-built modules. Update the caller at `app/generation/page.tsx` to use the new interface.

## Acceptance Criteria
- [ ] `POST /api/generate-ai-code-stream` accepts `{ modules: string[], instanceConfig: InstanceConfig }`
- [ ] Returns HTTP 400 with `{ error: "unknown_module", name: "<name>" }` for any module name not in `["store", "warehouse"]`
- [ ] For valid requests: streams Server-Sent Events (SSE) with assembly progress steps
- [ ] SSE event types: `step` (progress message), `done` (completion with output summary), `error` (failure)
- [ ] Assembly steps streamed: `validate_modules` → `copy_module_files` → `generate_navigation` → `apply_theme` → `done`
- [ ] The route reads module manifests from `modules/<name>/index.ts` to build the assembly plan
- [ ] Arbitrary freeform prompt generation is removed — the route no longer accepts a `prompt` field
- [ ] `app/generation/page.tsx` updated to call the new interface (passes `modules` + `instanceConfig` instead of a prompt)
- [ ] `app/generation/page.tsx` handles and displays SSE step events
- [ ] `tsc --noEmit` passes
- [ ] Integration test: POST with `modules: ["store"]` → streams all 5 step events → ends with `done`
- [ ] Integration test: POST with `modules: ["crm"]` → 400 response

## Technical Details

### New route interface:
```typescript
// app/api/generate-ai-code-stream/route.ts
import { NextRequest } from 'next/server'

const ALLOWED_MODULES = ['store', 'warehouse'] as const

export async function POST(req: NextRequest) {
  const { modules, instanceConfig } = await req.json()

  for (const name of modules) {
    if (!ALLOWED_MODULES.includes(name)) {
      return Response.json({ error: 'unknown_module', name }, { status: 400 })
    }
  }

  const encoder = new TextEncoder()
  const stream = new ReadableStream({
    async start(controller) {
      const send = (type: string, data: object) =>
        controller.enqueue(encoder.encode(`data: ${JSON.stringify({ type, ...data })}\n\n`))

      send('step', { message: 'Validating modules...', step: 1, total: 5 })
      // ... assembly steps
      send('done', { modules, instanceId: instanceConfig.instanceId })
      controller.close()
    }
  })

  return new Response(stream, {
    headers: { 'Content-Type': 'text/event-stream', 'Cache-Control': 'no-cache' }
  })
}
```
