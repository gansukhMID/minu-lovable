# Task: Build shared/ infrastructure (event-bus, db, auth, qr-payment adapter)

## Description

Create the `shared/` directory at repo root with four sub-packages that all modules depend on: an in-process event bus, a PostgreSQL database client, JWT auth middleware, and the QR payment adapter interface + stubs. This is the foundation that Store and Warehouse modules build on top of.

## Acceptance Criteria
- [ ] `shared/event-bus/index.ts` exports `publish(event, payload)` and `subscribe(event, handler)` with TypeScript types for all platform events
- [ ] Typed platform events defined: `order.created`, `stock.low`, `stock.insufficient`, `payment.received`
- [ ] `shared/db/index.ts` exports a PostgreSQL connection pool, `query(sql, params)`, and `withTransaction(fn)` helper
- [ ] `shared/auth/index.ts` exports Hono JWT middleware and `requireRole(role: string)` helper
- [ ] `shared/middleware/qr-payment/index.ts` exports `QRPaymentAdapter` interface
- [ ] `KhanBankAdapter` and `GolomtAdapter` classes implement `QRPaymentAdapter` with `throw new Error("not implemented")` + TODO comments
- [ ] All files are TypeScript strict — no `any` types
- [ ] Unit tests pass for event-bus (publish → subscriber called, multiple subscribers, unknown event is no-op)
- [ ] `tsc --noEmit` passes

## Technical Details

### shared/event-bus/index.ts
```typescript
// In-process EventEmitter wrapper for Phase 1; Phase 2 swaps to Redis pub/sub
import { EventEmitter } from 'events'

export type PlatformEvent =
  | { type: 'order.created'; payload: OrderCreatedPayload }
  | { type: 'stock.low'; payload: StockLowPayload }
  | { type: 'stock.insufficient'; payload: StockInsufficientPayload }
  | { type: 'payment.received'; payload: PaymentReceivedPayload }

export function publish<T extends PlatformEvent['type']>(
  event: T,
  payload: Extract<PlatformEvent, { type: T }>['payload']
): void

export function subscribe<T extends PlatformEvent['type']>(
  event: T,
  handler: (payload: Extract<PlatformEvent, { type: T }>['payload']) => void | Promise<void>
): () => void  // returns unsubscribe function
```
