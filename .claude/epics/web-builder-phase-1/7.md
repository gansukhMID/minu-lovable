# Task: HomePage module-selection onboarding screen

## Description

Replace the current `app/page.tsx` / `app/landing.tsx` hero section with a module-selection onboarding screen. Users choose Store and/or Warehouse before starting a builder session. The selection is passed to the generation page as query params. All UI uses shadcn/ui only (https://ui.shadcn.com/).

## Acceptance Criteria
- [ ] `app/page.tsx` renders a module-selection screen instead of the current landing/hero content
- [ ] Store and Warehouse each appear as a shadcn/ui `Card` with name, one-line description, and icon
- [ ] User can select one or both modules (multi-select, toggleable)
- [ ] "Start Building" `Button` is disabled (and visually grayed) when zero modules are selected
- [ ] Clicking "Start Building" navigates to `/generation?modules=store,warehouse` (or single module)
- [ ] Selected state is visually distinct (shadcn/ui `Card` with `ring-2 ring-primary` or similar)
- [ ] No custom CSS written — only Tailwind utility classes and shadcn/ui components
- [ ] Component is responsive (works on mobile and desktop)
- [ ] `tsc --noEmit` passes

## Technical Details

### Component sketch:
```tsx
// app/page.tsx
'use client'
import { useState } from 'react'
import { useRouter } from 'next/navigation'
import { Card, CardHeader, CardTitle, CardDescription } from '@/components/ui/card'
import { Button } from '@/components/ui/button'
import { Badge } from '@/components/ui/badge'

const MODULES = [
  { id: 'store', title: 'Store', description: 'Orders, products, and QR payment processing', icon: '🛒' },
  { id: 'warehouse', title: 'Warehouse', description: 'Stock levels and inventory movement tracking', icon: '📦' },
] as const

export default function HomePage() {
  const [selected, setSelected] = useState<string[]>([])
  const router = useRouter()

  const toggle = (id: string) =>
    setSelected(prev => prev.includes(id) ? prev.filter(m => m !== id) : [...prev, id])

  const handleStart = () =>
    router.push(`/generation?modules=${selected.join(',')}`)

  return (
    <main className="min-h-screen flex flex-col items-center justify-center gap-8 p-8">
      <h1 className="text-3xl font-bold">Choose your modules</h1>
      <div className="grid grid-cols-1 sm:grid-cols-2 gap-4 w-full max-w-2xl">
        {MODULES.map(mod => (
          <Card
            key={mod.id}
            onClick={() => toggle(mod.id)}
            className={`cursor-pointer transition-all ${selected.includes(mod.id) ? 'ring-2 ring-primary' : ''}`}
          >
            <CardHeader>
              <CardTitle>{mod.icon} {mod.title}</CardTitle>
              <CardDescription>{mod.description}</CardDescription>
            </CardHeader>
          </Card>
        ))}
      </div>
      <Button disabled={selected.length === 0} onClick={handleStart} size="lg">
        Start Building
      </Button>
    </main>
  )
}
```
