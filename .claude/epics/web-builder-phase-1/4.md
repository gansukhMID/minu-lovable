# Task: Build modules/store/ (orders, products, QR payment, shadcn/ui pages)

## Description

Build the Store module implementing the `Module` interface from PLATFORM.md §7. Covers the database schema, Hono API routes, event publishing, QR payment integration, and shadcn/ui frontend pages. The module must only import from `shared/` — never from other modules.

## Acceptance Criteria
- [ ] `modules/store/` implements the `Module` interface (name, version, routes, components, schema, events, middlewareNeeds)
- [ ] Database schema: `orders`, `order_items`, `products` tables with SQL migration file
- [ ] API routes working: `GET /store/orders`, `POST /store/orders`, `GET /store/orders/:id`, `PATCH /store/orders/:id/status`, `GET /store/products`, `POST /store/products`
- [ ] `POST /store/orders` publishes `order.created` event via `shared/event-bus`
- [ ] `order.created` payload includes `{ orderId, items: [{ productId, qty, unitPrice }], customerId, total }`
- [ ] QR payment: `POST /store/orders/:id/payment` creates a payment via `QRPaymentAdapter`, returns QR code URL
- [ ] Frontend: `OrdersPage` lists orders with status badges (shadcn/ui `DataTable`, `Badge`)
- [ ] Frontend: `NewOrderPage` creates an order with product selection (shadcn/ui `Form`, `Select`, `Input`)
- [ ] Frontend: `PaymentPage` shows QR code and polls payment status (shadcn/ui `Card`, `Dialog`)
- [ ] All frontend components use shadcn/ui only — https://ui.shadcn.com/
- [ ] Unit tests: order creation publishes correct event payload
- [ ] `tsc --noEmit` passes

### Orders schema (schema.sql excerpt):
```sql
CREATE TABLE orders (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  status TEXT NOT NULL DEFAULT 'pending' CHECK (status IN ('pending','paid','payment_hold','cancelled')),
  customer_id TEXT,
  total NUMERIC(12,2) NOT NULL,
  created_at TIMESTAMPTZ NOT NULL DEFAULT now()
);
CREATE TABLE order_items (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  order_id UUID NOT NULL REFERENCES orders(id),
  product_id UUID NOT NULL,
  qty INT NOT NULL CHECK (qty > 0),
  unit_price NUMERIC(12,2) NOT NULL
);
CREATE TABLE products (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  name TEXT NOT NULL,
  price NUMERIC(12,2) NOT NULL,
  sku TEXT UNIQUE,
  active BOOLEAN NOT NULL DEFAULT true
);
```
